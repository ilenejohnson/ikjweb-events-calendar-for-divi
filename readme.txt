Extension of Tribe Events Calendar - shows recent Events


== Description ==

IKJ Events Calendar For Divi is a WordPress plugin extends the Tribe Events Calendar.  

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder.
2. Go to the **Plugins** page and activate the plugin.


== Frequently Asked Questions == 

= Are other plugins required?  

Yes, you will need the Events Calendar by Tribe as this plugin is an extention and either have a DIVI theme or the DIVI plugin
